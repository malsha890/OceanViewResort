# Ocean View Resort - Hotel Management System

A Java Servlet MVC web application for managing hotel operations.

## Features
- Staff Authentication with BCrypt password hashing
- Room Management (Add, Update, Delete, Search)
- Reservation Management with date-overlap validation
- Billing with 5% tax and 10% service charge
- Email confirmation on reservation
- Hotel Reports with monthly charts
- Admin Dashboard for staff management

## Tech Stack
- Java Servlets (MVC pattern)
- JSP (Views)
- MySQL (Database)
- Apache Tomcat 10
- Maven (Build tool)
- JUnit 5 + Mockito (Unit testing)

## How to Run
1. Clone the repository
2. Import into Eclipse as Maven project
3. Configure MySQL database
4. Deploy on Apache Tomcat 10
5. Access at: http://localhost:8080/OceanViewResort
